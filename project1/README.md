# Contact v1 · 红蓝对决

基于纯 HTML/CSS/JavaScript 的 Canvas 回合制战术对战游戏。零外部依赖，单文件运行。

**文件：** `contactv1.html`

## 快速开始

直接在浏览器打开 `contactv1.html` 即可：

```bash
start contactv1.html
```

无需安装、无需构建、无需服务器。

## 游戏模式

| 模式 | 说明 |
|------|------|
| **人机对战** | 玩家控制红队，AI 控制蓝队 |
| **双人对战** | 两名玩家轮流操作红蓝双方 |

## 游戏规则

- 每队默认 5 人（可调 3-7 人）
- 回合制，每回合选一名队员行动
- **操作**：点击选中队员 → 鼠标瞄准 → 空格蓄力（力量条来回摆动）→ 松开射击
- **子弹**：从队员圆心发射，**穿透敌人即击杀**，队员移动到子弹落点
- **弹性碰撞**：子弹遇到障碍物、边界或队友会**反弹**，不会停止
- **胜利条件**：全歼对方所有队员

## 功能特性

| 功能 | 说明 |
|------|------|
| 双模式 | 人机对战 / 双人对战 |
| 多地图 | 空地、经典、迷宫、竞技场（含障碍物） |
| 弹性碰撞 | 子弹遇边界/障碍物/队友反弹 |
| 地图操作 | 滚轮缩放、拖拽平移 |
| 暂停菜单 | Esc 打开，可继续/统计/设置/退出并保存 |
| 多存档位 | 最多 10 个存档，可读取任意存档 |
| 统计数据 | 场次/胜率/KD比，持久化保存 |
| 自适应窗口 | 画布自动适配浏览器大小 |
| 简约现代风 | 纯色背景 + 彩色圆形队员 |

## 操作说明

| 操作 | 按键/鼠标 |
|------|----------|
| 选中队员 | 左键点击当前回合方的队员 |
| 切换选中 | 左键点击其他队员 |
| 瞄准 | 移动鼠标（显示方向指示线） |
| 蓄力发射 | 按住空格 → 松开 |
| 暂停/继续 | Esc |
| 地图拖拽 | 左键空地拖拽 / 中/右键拖拽 |
| 地图缩放 | 滚轮 |
| 返回菜单 | 游戏结束 Esc |

## 设置选项

| 设置 | 范围 | 说明 |
|------|------|------|
| 队伍人数 | 3-7 | 每队队员数量 |
| 蓄力速度 | 0.5-2.0 | 力量条摆动速度 |
| 最大距离 | 50%-150% | 子弹最远飞行距离 |
| 地图 | 4种 | 切换不同障碍物布局 |

## 代码架构

单文件 `contactv1.html`，结构如下：

- **CSS (~20 行)** — 全屏 Canvas 样式
- **JavaScript (~1000+ 行)** — IIFE 自执行函数
  - 屏幕系统：menu / game / settings / statistics / load
  - 回合生命周期：select → aim → charge → bullet → resolve
  - 摄像机系统：平移 + 缩放，坐标自动转换
  - 弹性碰撞：边界/矩形障碍物/队友反弹物理
  - AI 逻辑：瞄准最近敌人 + 随机偏移 + 随机蓄力
  - 多存档位：localStorage 存储最多 10 个存档
  - 统计持久化：场次、胜率、KD 比

## 关键技术点

- **碰撞系统**：线段与圆的碰撞检测（击杀）、矩形碰撞检测（障碍物反弹）、圆心方向反弹（队友）
- **反弹物理**：世界边界 → 速度分量取反；障碍物 → 判断最近边反弹；队友 → 从圆心方向反射
- **摄像机**：`screenToWorld()` 坐标转换，`withCamera()` 统一渲染变换
- **蓄力机制**：力量值在 0-100 之间来回摆动，松开时定格
